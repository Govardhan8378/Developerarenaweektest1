# User Information Program

A simple Python script that asks for the user's **name**, **age**, and **hobby**, then displays a friendly welcome message.

## Setup

1. Install Python 3.
2. Save the script as `personal_info.py`.
3. Run:

```bash
python personal_info.py
```

## Code

```python
name = input("Enter your name: ")
age = input("Enter your age: ")
hobby = input("Enter your hobby: ")

print(f"You are {age} years old and {hobby}.")
```

## Sample Output

```
Enter your name: Govardhan
Enter your age: 21
Enter your hobby: Reading
You are 22 years old and  Reading.
```
